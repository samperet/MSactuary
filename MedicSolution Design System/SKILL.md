If creating visual artifacts (slides, mocks, throwaway prototypes, etc.), copy assets out and create static HTML files for the user to view. The brand is Swedish, engineering-led, navy-anchored, and quietly confident — match that tone. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

Key things to honour (full detail in README):
- Brand colour is **Navy 700 `#0b426a`**, taken directly from the MS+ logo. Indigo `#3837cc` is a sparing digital accent, not a co-primary.
- Type: display = Familjen Grotesk, body = IBM Plex Sans, mono = IBM Plex Mono (Google Fonts substitutes — see README).
- Voice is engineering-led, plural ("we"), British English, no emoji, no exclamation marks, numerics carry the brag.
- Visual language is flat-ish, square-ish corners, restrained shadows, generous white space, real product photography. No gradients, no patterns, no scale-on-press.
- Iconography is almost absent. Default to no icon. Lucide stroke-only at 1.5 px if you must.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts or production code, depending on the need.
